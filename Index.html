For this week, I learn recap on what I learn from week 5 JavaScript basic

  1. Literal Object
    • Has no constructors
    • Cannot be inherited
    • A colon separates the key/value pair

    • Has constructors

Week 7

For this week, I learn more about JavaScript such as Document Object Model(DOM), Event Handling & Validation and Local Storage.

  1. DOM allows you to access element by using:
    • document.getElementById(); to get an element using the id its using
    • document.querySelector(); to get the first matching element
    • document.getElementsByClassName to get all element that is using that class name
    • document.querySelectorAll() to get all matching element

    Using methods such as:
    • <element>.className to get the class name of an element or change the class name
    • <element>.classList to remove or add a class to an element
    • <element>.textContent to set content in an element
    • <element>.innerhtml to retrieve content or modification of new content in an element


  2. Event Handling
    • AddEventListener("click",function()) when an element is clicked and calls the function
    • AddEventListener("mouseover",function()) when cursor hovers over an element and calls the function


  3. Local Storage
    • localStorage.setItem(key, value) Stores the value
    • localStorage.getItem(key) to get the values using the key
      • OR
    • localStorage.key = value to store the value
    • localStorage.key to get the value if there is any

    • localStorage.length to get the number of items stored
    • localStorage.removeItem(key) to remove item in storage
    • localStorage.clear() to clear all items in storage

Week12

For this week, I learn about NoSQL and a NoSQL database, restdb.

  1. NoSQL
    • When you have high volume of data
    • Document oriented with key : value (key pair values) instead of table based which SQL database uses
    • Flexible as it does not have schema


  2. Restdb
    • Get Method

      • This method is used to get documents from the collection

    • Post Method

      • Post method is use to post a new docuement to the collection

    • Put Method

      • This method updates a document in the collection

    • Delete Method

      • Deletes a document in the collection
